## Conteúdo Técnico
VAR
```JS
var x = 5;
var y = 6;
var z = x + y;
```
LET
```JS
let x = 5;
let y = 6;
let z = x + y;
```
CONST
```JS
const x = 5;
const y = 6;
const z = x + y;
```
STRING
```JS
const single = 'Single quotes';
const double = "Double quotes";
const backtick = `Backtick`;

console.log(single);
console.log(double);
console.log(backtick);
```
